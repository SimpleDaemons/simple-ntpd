simple-ntpd - Read Me

Quick Start

After installation, simple-ntpd will be available as a system service.

Configuration

Edit the configuration file:

/etc/simple-ntpd/simple-ntpd.conf

Service Management

Start the service:

sudo systemctl start simple-ntpd

Enable at boot:

sudo systemctl enable simple-ntpd

Full documentation is available at:
